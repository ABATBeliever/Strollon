## 概要

Strollonは、PySide6とQtWebEngineを利用したマルチプラットフォームのウェブブラウザです。  
過去のInternetStrollerシリーズの設計思想と、VELA Praxisの拡張性を合流させた、新しいブラウザです。

## 主要機能

- Chromium ベースのレンダリング
- Windows/Linuxで動作

- タブブラウジング
- わかりやすく、洗練された軽量システムデザイン
- 縦タブ
- 同じメジャーバージョンでは互換性を維持
- 軽量な設計
- XDGに準拠、またはポータブルもOK
